# Modern E-Commerce Website

This project is a modern and stylish web application providing essential e-commerce features with a focus on user experience. Built with HTML, CSS, and Vanilla JavaScript.

- **Dynamic Product Listing:** Showcase featured products on the homepage.
- **Detailed Product Pages:** View product descriptions, prices, and related product suggestions.
- **Advanced Search System:**
    - Site-wide accessible search bar.
    - Trigger search via the "Enter" key or by clicking the magnifying glass icon.
    - Dedicated search results page (`search-results.html`).
- **Wishlist:** Add/remove products to favorites with persistent storage using localStorage.
- **Cart Management:** Add products, increase/decrease quantities, and clear items.
- **Checkout Process:** Order summary and demo mode payment completion.
- **Toast Notifications:** Sleek feedback messages for user interactions.
- **Responsive Design:** Fully compatible with mobile, tablet, and desktop devices.
- **Hamburger Menu:** Optimized navigation for mobile users.
